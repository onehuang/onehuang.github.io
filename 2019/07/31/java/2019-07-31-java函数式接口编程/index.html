    1. The java.util.function Package

      在java8中加入 java.util.function包，主要是基础下面四个模式：

      模式 参数 返回值 描述
      Predicate 是 Boolean 测试参数并且返回true 或 false
      Function 是 是 映射一个类型到其他类型
      Consumer 是 否 消费输入，没有返回值信息
      Supplier 否 是 产生数据，没有输入信息

    2. Predicate 接口

      定义一个整形类型的Predicate:
